
## Structures!
- Similar to NamedTuple in python (Collections) and class in Java

```C
//Looks like:
struct structName {
	type item1;
	type item2;
};
//Example:
struct car {
	char make[40];
	char model[40];
	unsigned int year;
	float price;
} chan, woit; //<- this declares 2 car variables named chan and woit
```
